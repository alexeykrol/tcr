**ВЫСШИЕ РОЛИ**  

| РОЛЬ | ОСНОВНЫЕ ПРИЗНАКИ |
| --------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Лидер | Генерирует идеи и объединяет людей вокруг какой-либо масштабной задачи, ведет за собой. Управляет больше на уровне эмоций и сердца, чем здравого смысла. Обладает виденьем, может указывать направление развития, вдохновлять людей. Бесстрашен, со всеми находит общий язык, умеет нравиться. |
| Политик | Управляет людьми в рамках иерархии. Заводит и поддерживает связи. Лоббирует, угрожает, соблазняет, добивается результата. Вступает в стратегические союзы и альянсы. Следит за поддержанием баланса сил, обеспечивает интересы людей и группировок, которым он принадлежит. Вербует, столбит сторонников. Ничего не создает. Умеет подбирать кадры и управлять ими. Влияет на людей, способен нравиться. |
| Финансист | Управляет финансовыми потоками, контролирует ресурсы, настроен на снижение затрат при увеличении прибыли, своей главной задачей видит рост капитализации. Управляет людьми, влияет на них, отстаивает интересы своей организации. |
  
  
**СРЕДНИЕ РОЛИ**  

| РОЛЬ | ОСНОВНЫЕ ПРИЗНАКИ |
| ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Продюсер | Ищет идеи, людей, ресурсы. Связывает все воедино. Обеспечивает реализацию идей. Создает команды и управляет ими. Созидает, но в проектном смысле. Результат деятельности — претворение идеи в жизнь. |
| Управленец | Управляет людьми, организует их на выполнение различных задач, анализирует ситуацию, распределяет ресурсы, контролирует ход работы. Мотивирует, поощряет, наказывает. Создает ценность через управление людьми. Результат деятельности — реализация проекта в срок. |
| Торговец | Совершает выгодные сделки, находит клиентов, договаривается об условиях. Коммуникатор, переговорщик, манипулятор. Влияет на людей, понимает, как зарабатывать деньги. Результат деятельности — сделка. |
| Советник | Наблюдает, анализирует, делает выводы, на их основе дает советы. Обрисовывает перспективы и альтернативы, привлекает экспертов, лоббирует. Хорошо разбирается в предмете, но сам ничего не производит. Не управляет другими людьми, но может оказывать большое влияние. Результат — изменение видения людей, их оценки происходящего и образа действия. |
| Преподаватель | Учит других людей. Не управляет, но может оказывать минимальное влияние. Результат — освоение людьми новых знаний и навыков, иногда смена виденья. |
  
  
**НИЗШИЕ РОЛИ**  

| РОЛЬ | ОСНОВНЫЕ ПРИЗНАКИ |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Специалист | Работает головой. Реализует проекты, находит подрядчиков, принимает работу, анализирует, проектирует, планирует, создает, консультирует. Это ученый, аналитик, бухгалтер, редактор, IT-специалист и т. п. Производит операции с нематериальными объектами, генерирует идеи, решает прикладные задачи, отвечает на вопрос «КАК», который ставят руководители. Результат деятельности — решение конкретной задачи. |
| Исполнитель | Работает руками, производит операции с материальными объектами: обработка материала, сборка устройств, упаковка продукции, управление машинами, выдача и прием материальных объектов, транспортировка чего или кого-либо, охрана и пр. Не управляет людьми. Результат деятельности — успешные манипуляции с материальными объектами. |
